Evidencia de aprendizaje. Programa con conexión a base de datos.
Como parte de la evaluación de esta unidad, es necesario realizar un programa  que 
resuelva una problemática de tu entorno, el cual requiera manipular información con 
conexión base de datos.

El propósito es realizar un programa que inserte, modifique, elimine y muestre información 
de una base de datos.

Instrucciones:

1.  Ubica un problema a resolver de tu libre elección, (Ejemplos: control de inventarío, 
cobro de una tienda, venta de pasajes de autobús, control escolar, etc.) y  redacta
en un procesador de textos el problema.

2.   Redacta  cómo  resolverías el planteamiento del problema del punto uno o
puedes utilizar la misma problemática de la practica 3, solo deberás  ampliar la 
descripción, tomando en cuenta que este programa debe insertar, modificar, 
eliminar y mostrar información.

3.  Una vez resuelto el problema identifica los componentes necesarios para
resolver el problema planteado en el punto anterior subrayándolos de color verde.
4.  Posteriormente crea lo siguiente:

  El script que construirá la base de datos con que se conectará el programa.
  Un proyecto en NetBeans e incluye una interfaz gráfica de usuario para el 
programa.

  La clase de conexión y manipulación de datos, así como las clases para el 
control del programa dentro del proyecto NetBeans creado en el punto 
anterior.
